# XSmall-HMI-SCADA_input_output
Библиотека должна реализовывать визуальные объекты (текст,поле ввода, кнопка) и изменять их параметры в зависимости значений в базе переменных. Необходимо программу на C++17, модульные тесты при помощи gtest.    В качестве библиотеки для работы должна использоваться библиотека SFML. Программа должна иметь MIT лицензию
